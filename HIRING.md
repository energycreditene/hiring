# We’re Hiring – ENE Gaming/Ticketing Platform (Vendor RFP)

**Energy Credit (ENE)** seeks a studio/agency to build a lightweight **gaming/ticketing web platform** integrated with ENE (ERC-20). Goal: enable venues to issue/redeem play credits and accept ENE.

## Phase 1 (8–12 weeks)
- Web app (React/Next.js), roles: **admin / venue / player**
- WalletConnect + ENE payments (buy credits, redeem)
- Ticketing: issue QR codes, validate, history
- Admin panel for venues: redemptions & simple metrics
- API (Node/Nest), DB (Postgres/Prisma), hosted (Vercel/Netlify)
- QA, docs, handover

### Nice to have (Phase 2)
- Mobile wrapper (Expo), rewards, POS/offline mode, charts

## Deliverables
- Source code (license to ENE), infra basics (env/scripts), README runbook
- Staging URL + production handoff
- Milestone demo videos

## Proposal format (submit via Issue template)
- Team intro + links
- Approach/architecture (short diagram)
- Timeline by milestones
- Fixed price per milestone (USD)
- 2–3 case studies
- Security approach
- Contact + timezone

**Budget target (Phase 1):** $15k–$40k USD · **Start:** ASAP · **Timezones:** Americas/Europe
